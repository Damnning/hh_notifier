# HH Notifier — Telegram-бот для мониторинга вакансий

Telegram-бот для автоматического поиска и уведомления о новых вакансиях с портала [HeadHunter](https://hh.ru/).

## Возможности

- **Автоматический мониторинг** — бот проверяет новые вакансии каждые 10 минут
- **Умная фильтрация** — поиск по ключевым навыкам: Data Scientist, ML Engineer, Python Backend, Data Analyst
- **Фильтр по опыту** — «Без опыта» или «1–3 года»
- **Географический фильтр** — Воронеж или удалённая работа (вся Россия)
- **Чёрный список** — автоматическое отсеивание вакансий не по профилю (Senior, Fullstack, Java, QA и др.)
- **Персистентность** — сохранение списка просмотренных вакансий между запусками

## Технологический стек

- **Python 3.10+**
- [aiogram](https://aiogram.dev/) — асинхронный фреймворк для Telegram-бота
- [aiohttp](https://docs.aiohttp.org/) — асинхронный HTTP-клиент для API hh.ru
- Docker + Docker Compose

## Установка и запуск

### Локальный запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Настройте токен бота в [`bot.py`](bot.py:11):

```python
API_TOKEN = 'ВАШ_TELEGRAM_BOT_TOKEN'
```

3. Разрешённые пользователи в [`bot.py`](bot.py:15):

```python
ALLOWED_USERS = [ВАШ_TELEGRAM_ID]
```

4. Запустите бота:

```bash
python bot.py
```

### Запуск в Docker

1. Соберите и запустите контейнер:

```bash
docker-compose up -d
```

2. Для пересборки:

```bash
docker-compose up -d --build
```

## Использование

1. Найдите бота в Telegram по имени пользователя
2. Отправьте команду `/start`
3. Бот начнёт мониторинг и пришлёт уведомление о запуске
4. При появлении новых вакансий вы получите уведомление с:
   - Названием вакансии
   - Требуемым опытом
   - Местоположением (Воронеж / Удалёнка)
   - Названием работодателя
   - Зарплатой
   - Ссылкой на вакансию

## Настройка поиска

Параметры поиска можно изменить в [`bot.py`](bot.py):

### Ключевые слова поиска (строки 25–47)

```python
SEARCH_QUERIES = [
    'Data Scientist',
    'ML Engineer',
    'Machine Learning',
    ...
]
```

### Черный список слов (строки 50–75)

```python
EXCLUDED_WORDS = [
    'Senior', 'Fullstack', 'Java', 'QA', ...
]
```

### Интервал проверки (строка 12)

```python
CHECK_INTERVAL = 10 * 60  # 10 минут в секундах
```

## Требования к API

Бот использует официальное API hh.ru:

- **Base URL:** `https://api.hh.ru/vacancies`
- **User-Agent:** `MyTelegramBot/3.0 (danning600@gmail.com)`

Ограничения API: не более 20 запросов в секунду. Бот автоматически соблюдает паузы между запросами.
